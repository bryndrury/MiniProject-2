	To build and deploy:
docker-compose up

	To just build:
For all Images:
docker-compose build

For individual: -> (i.e. service = worker)
docker-compose build <service>


	Running:
For RabbitMQ Service: -> (attached)
docker-compose up rabbitmq

For manager: -> (attached)
docker-compose run --rm --name worker-manager manager

For worker: -> (attached)
docker-compose run --rm worker
